Time

Time objects

class Instant:
Denotes a specific instant in time.

func new(sec, nsec): Creates a new time instant sec seconds + nsec nanoseconds since the local epoch.

func current(): Returns the current system time as an Instant. This function returns conventional wall clock time.

func current_mono(): Returns the current system time as an Instant. This function guarantees to always return a monotonically increasing value; it is useful for timing purposes, as it is not skewed to reflect the realities of the outside world. Note: not all platforms support this functionality (e.g. OS X), in which case the value returned will be the same as current.
